# Task Manager - Frontend

A full stack Task Manager app built with React, React Router, and TypeScript.

## Features
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Tasks persist in MongoDB via backend API

## Tech Stack
- React
- React Router
- TypeScript
- Vite

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. App runs on `http://localhost:5173`

## Backend
This frontend connects to [task-manager-backend](https://github.com/dhanushquinn-collab/task-manager-backend) running on `http://localhost:3000`

Make sure the backend server is running for the app to work properly.
